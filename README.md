# FireDragon  
  
**FireDragon is an MCBE Server-Proxy for Podrum**

# Good things
**1.** Fast.

**2.** Secure.

**3.** Easy API to write Plugins in Python.

# FAQ
- Can i Contribute?
> Yes you can.

- With what for Server softwares FireDragon work?
> It is in planing that it works with **every** software.

- Do we have a Discord?
> Yes we have [Discord](https://discord.gg/dbBQrNmFFy)

- Do we have a Documentation?
> Not yet.

## Running:
Linux and MacOS (Unix systems):
```sh
python3 FireDragon
```
Or you can use Powershell.

Windows:
```batch
python FireDragon
```
